# Bless Us - Wedding Invitation

A beautiful, responsive wedding invitation web application for **Danni & Kichu**.

## 📅 Save the Date
- **Date:** May 21, 2026
- **Location:** Jaffna, Sri Lanka

---

## ✨ Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Countdown Timer:** Real-time countdown to the big day.
- **Event Details:** Comprehensive information about the ceremony and reception with Google Maps integration.
- **RSVP System:** Interactive response form with meal preferences and guest counting.
- **Photo Gallery:** A sneak peek at the couple's special moments.
- **Modern UI:** Built with Bootstrap 5 and custom CSS for a premium feel.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Frameworks:** [Bootstrap 5.3](https://getbootstrap.com/)
- **Icons:** [Font Awesome 6](https://fontawesome.com/)
- **Fonts:** Custom typography for a wedding aesthetic.

## 📂 Project Structure

```text
Bless us/
├── index.html          # Homepage with hero section and countdown
├── Eventdetail.html    # Ceremony and Reception details
├── Response.html       # RSVP / Response form page
├── style.css           # Custom styling and design tokens
├── script.js           # Countdown logic and form handling
└── Images/             # Project images and assets
    └── logo.png        # Project logo
```

## 🚀 Getting Started

1. **Clone/Download** the repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the pages using the navigation bar.
